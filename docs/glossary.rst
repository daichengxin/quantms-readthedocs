Glossary
========

.. glossary::

  DDA
    Data-Dependent Acquisition. MS/MS fragmentation triggered by detected precursor ions.

  DIA
    Data-Independent Acquisition. Systematically fragments wide m/z windows, then deconvolves signals computationally.

  SDRF
    Sample and Data Relationship Format. A tabular file describing experimental design and file locations.

  mzML
    HUPO-PSI XML format for raw spectra.

  mzTab
    HUPO-PSI tabular format for identification and quantification results.

  Contaminants
    Common lab protein sequences included in FASTA with a ``CONTAM_`` prefix for filtering/QC.

  Decoy
    Artificial protein sequences (e.g., reversed) used for FDR estimation in targetâ€“decoy approaches.

  Entrapment
    Proteins from an organism not present in the sample, appended to assess empirical FDR.
